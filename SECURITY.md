# Security Policy

Rased NDR v0.1.1 performs passive analysis of packet-capture files supplied by the user.

## Safe use

- Analyze only captures you own or are authorized to inspect.
- Do not upload captures containing sensitive credentials or private user data to untrusted systems.
- The current version does not perform exploitation, scanning, or live packet capture.

## Reporting a vulnerability

Please open a private GitHub Security Advisory instead of a public issue.
